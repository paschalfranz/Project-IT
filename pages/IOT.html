The equipments that were used in this project are: BMP280 sensor, BH1750 sensor, Breadboard, Orangepi, Wires, LED, resistors, Driver ULN2003 and pushbuttons. And the application terminal that i used was visual studio code.

The End Goals:

  1. Show temperature and lux values on terminal
  2. Show temperature and lux values on cloud dashboard
  3. Set goal for temperature and lux and show on dashboard
  4. Control light intensity
  5. Control temperature
  6. Automatically adjust light and temperature
